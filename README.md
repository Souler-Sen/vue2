# vue2空白项目
  如有需要可直接拉取使用
  
## 配置内容：
  VueX、VueRouter、Axios、Element-UI、Scss、Eslint

## Project setup
```
npm install
```

### Compiles and hot-reloads for development

项目本地运行
```
npm run serve
```

### Compiles and minifies for production

测试站点打包
```
npm run build:test
```

正式站点打包
```
npm run build:prod
```


### Lints and fixes files
```
npm run lint
```
